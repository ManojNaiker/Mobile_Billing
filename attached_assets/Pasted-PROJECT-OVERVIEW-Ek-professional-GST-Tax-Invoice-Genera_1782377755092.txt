PROJECT OVERVIEW
Ek professional GST Tax Invoice Generator web application banao jo Indian small businesses ke liye ho. App ka naam BillEase hai. Yeh app Tally jaisi look mein GST invoices create, manage aur print kar sake.

TECH STACK
Frontend  : React + Vite + TypeScript + Tailwind CSS + shadcn/ui
Routing   : wouter (React Router ki jagah)
Backend   : Node.js + Express 5 + TypeScript
Database  : PostgreSQL + Drizzle ORM
Validation: Zod
Charts    : Recharts
Package   : pnpm workspaces (monorepo)

BUSINESS LOGIC
GST Calculation
1. Rate × Quantity = Base Amount
2. Base Amount × (1 - Discount%/100) = Taxable Value
3. If intra-state:
   CGST = Taxable Value × (Tax%/2) / 100
   SGST = Taxable Value × (Tax%/2) / 100
   IGST = 0
4. If inter-state:
   CGST = 0, SGST = 0
   IGST = Taxable Value × Tax% / 100
5. Total = Taxable Value + CGST + SGST + IGST
6. Grand Total = Sum of all item Totals
7. Round to 2 decimal places

Amount in Words (Indian System)
1 lakh = 1,00,000
1 crore = 1,00,00,000
Example: ₹1,25,430 → "Rupees One Lakh Twenty-Five Thousand Four Hundred Thirty Only"

Invoice Number Auto-generation
Format: {prefix}-{counter padded to 4 digits}
Example: INV-0001, INV-0002, BILL-0023
Counter increments on every new invoice (including duplicates)
Stored in company table's invoice_counter column

Duplicate Invoice
- Original invoice copy karo
- New invoice number assign karo (next counter)
- Date: aaj ki date set karo
- Status: "draft" set karo
- Sab doosra data same rakho

Invoice Number	Unique, required
Invoice Date	Required, valid date
Buyer Name	Required
Buyer Address	Required
Company Name	Required
Company Email	Required, valid email
Item Rate	> 0
Item Quantity	> 0
